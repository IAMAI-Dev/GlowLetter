# GlowLetter

“绿荧来信”微信小程序，使用原生 WXML、WXSS、JavaScript 与微信云开发。当前版本已将 OpenDesign 的 10 个原型页面完整转换为可运行的演示流程。

## 在微信开发者工具中查看

1. 选择“导入项目”，项目目录指向仓库根目录。
2. 开发者工具会根据 `project.config.json` 自动使用 `miniprogram/` 作为小程序目录。
3. 点击“编译”，即可从启动页进入首页；检测流程支持使用内置演示图完整走查。

当前分析结果为本地模拟数据，不代表真实实验或污染检测结论。OpenDesign 原型仅作为设计参考，位于被 Git 忽略的 `opendesign/` 目录中。
